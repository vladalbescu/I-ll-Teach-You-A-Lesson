1. What are functions?

Functions let us define a code of block which we can call and execute whenever we want. This helps us to write less and keep our code DRY.

Declaring a function consists of the function keyword followed by:

  1. the function's name (it is recommended to use a descriptive name which starts with a verb and uses camel case)
  2. a pair of closed parentheses
  3. a pair curly brackets within which we can write what the function will do (known as the function's body)
