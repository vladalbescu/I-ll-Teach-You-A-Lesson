2. Inputs and outputs

Optionally, a function can have one or multiple inputs (called parameters) and an output (or return value).

When we call a function that has parameters, we must give it some values inside the parentheses. These values are called arguments.

The following function has two parameters, x and y, and returns their sum. When we call it, we give it two arguments, 5 and 3, corresponding to its two parameters. The variable mySum is assigned the function's return value, which will be 8.
