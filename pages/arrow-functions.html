8. Arrow Functions

An arrow function is a shorter alternative to writing functions.

Instead of writing a function expression like this: 

let myFunction = function(x) { return x + 1 }
We can write it like this using arrow functions: 
let myFunction = function() => x + 1

Arrow functions also act different when it comes to the this keyword. Usually, the this keyword refers to the object that calls a function. However, arrow functions, inherit this from their parent scope.
